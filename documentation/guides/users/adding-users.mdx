---
title: "Removing users"
description: "To remove a user from your account:"
---

1. Go to **Settings** \> **Members**
2. Find the user in the members table
3. Click the remove icon (trash) next to their name
4. Confirm the removal

<Warning>
  Removing a user immediately revokes their access to your Vayu account. This action cannot be undone.
</Warning>
